# 🌀 UR5 ⚙️ - TypeScript Platform 🚀

### Dependences 🌜
1. Yarn 1.6 or above
2. NodeJS 9.0 or above
3. TypeScript 2.6 or above (in global)
4. Mocha 5.1 or above (in global)


### Hardware 🔥

1. PolyScope 1.8
2. URController 1.8

### Install 💦 
1. Create `config.ts`
2. Copy class for `config.example.ts` to `config.ts`
3. Set `IP` and `Port` Robot
4. `yarn build`
5. `yarn start`
6. You are perfect! 💞

### Run test ☃️

1. `yarn build`
2. `yarn test`
3. You are wonderful! ✨
